The Document Object Model

  • The DOM is created by the browser when the HTML Document loads inside it
  • Once the HTML Document is loaded in the browser, the browser creates an object which models this document. This object is called the Document Object
  • It contains many different properties about the HTML Document and many different methods that we can use to interact with the external document
  • Whenever we want to do something like removing an element from the browser, we do that via the Document Object
